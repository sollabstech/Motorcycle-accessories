# Motorcycle-accessories

Velocity Armor — a motorcycle accessories e-commerce homepage built with Next.js (App Router), TypeScript, Tailwind CSS v4, and Framer Motion.

The brand color palette (`--color-primary`, `--color-accent`, etc.) is derived automatically at runtime by sampling `public/logo.png` on a canvas — see `src/lib/palette.ts`.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```
